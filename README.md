## Tought process

- install the require modules such as nextjs, zustand
- folder structure
    - components
        - resulable components
    - pages
        - new pages such as product listing, product detail
    - hooks
        - resuable custom hooks
    - stores

    - utils
        - custom helper functions
    - styles
        - styling css, currently global css
    - __tests__
        - unit testing

        - end to end testing
